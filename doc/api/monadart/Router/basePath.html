String basePath
read / write

A string to be prepended to the beginning of every route this Router manages.

For example, if this Router's base route is "/api" and the routes "/users/" and "/posts" are added, the actual routes will be "/api/users" and "/api/posts". This property MUST be set prior to adding routes, an exception will be thrown otherwise.
