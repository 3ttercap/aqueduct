A abstract class that concrete subclasses will implement to provide request handling behavior.

Applications set up HTTP(s) listeners, but do not do anything with them. The behavior of how an application responds to requests is defined by its ApplicationPipeline. Instances of this class must implement the handleRequest method from RequestHandler - this is the entry point of all requests into this pipeline.
