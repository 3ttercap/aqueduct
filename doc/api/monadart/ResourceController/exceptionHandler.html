Function exceptionHandler
read / write

The exception handler for a request handler method that generates an HTTP error response.

The default handler will always respond to the HTTP request with a 500 status code. There are other exception handlers involved in the process of handling a request, but once execution enters a handler method (one decorated with Route), this exception handler is in place.
